список папок:
1. css - хранит стили сайта
2. js - файлы javascript
3. img - папка хранения картинок
список фалов:
шаблон состоит из 4 файлов
1. header.php - отвечает за шапку нашего сайта, включает в себя тег <head></head>
2. index.php - выводит основной контент сайта
3. sidebar.php - файл отвечает за сайдбар (боковая панель) сайта
4. footer.php - подвал сайта
обязательные системные файлы
5. style.css - файл стилей, который в обязательном порядке находится в корне шаблона и содержит информацию о теме
6. functions.php - мозг WordPress, системный файл настройки CMS
обязательные стандартные файлы
7. 404.php - файл шаблона который в будущем будет выводить страницу ошибки 404.
8. page.php - файл который отвечает за вывод страниц, например "О компании"
9. single.php - вывод полной новости
10. search.php - вывод результатов поиска
11. searchform.php - все манипуляции с формой поиска проделываются в этом файле
12. archive.php - выводит архив блога
13. comments.php - данный файл отвечает за вид комментариев
14. wiki_customize.php - в файле прописаны настройки сайта которыми можно управлять через админку
